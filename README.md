## How to Run

### Prerequisites
- Python 3.6 or higher.

### Steps to Run
1. Clone the repository:

    ```bash
    git clone https://github.com/bc889/parse-ports.git
    ```

2. Navigate to the project directory:

    ```bash
    cd parse-ports
    ```

3. Run the script using the following command:

    ```bash
    python main.py
    ```
